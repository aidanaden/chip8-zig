# Chip8 in zig

## Install

MacOS is the only tested platform, everything else should work tho (no promises xd)

### Brew

```sh
brew install aidanaden/tools/chip8-zig
```

### Nix

```sh
nix run github:aidanaden/chip8-zig
```
